# PRD: 눈보라 속 여동생 찾기

> 감각만으로 길을 찾는 미니멀 탐색 게임

---

## 1. 개요

### 1.1 제품 비전
UI, 미니맵, 퀘스트 마커 없이 **소리와 색감만으로** 눈보라 속 여동생을 찾는 1인칭 탐색 게임. 걷고 찾는 행위 자체가 재미가 되는 감각 중심의 경험을 제공한다.

### 1.2 핵심 가치
- **감각적 몰입**: 시각 정보를 최소화하고 청각/색감 변화로 방향 감각 전달
- **감정 곡선**: 불안(시작) → 집중(탐색) → 안도(발견)
- **접근성**: 링크 하나로 즉시 플레이, 1~5분 내 클리어

---

## 2. 문제 정의

| 기존 문제 | 본 게임의 해결책 |
|-----------|------------------|
| UI/미니맵/마커로 인한 몰입 분산 | 모든 UI 제거, 감각 단서만 제공 |
| 정보 과잉으로 탐색의 긴장감 상실 | 눈보라로 시야 제한, 방향 감각 교란 |
| 전투/성장 없이 재미를 주기 어려움 | 찾는 행위 자체의 긴장감과 발견의 카타르시스 |

---

## 3. 타깃 사용자

### 3.1 주요 타깃
- 짧은 시간에 감정적 몰입을 원하는 캐주얼 게이머
- 전투 없는 실험적/아트 게임에 관심 있는 유저
- SNS에서 독특한 게임을 공유하고 싶은 유저

### 3.2 사용자 시나리오
```
1. 플레이어가 링크를 클릭해 게임 시작
2. 눈보라 속에서 아무런 UI 없이 방향 감각을 잃음
3. 희미한 울음소리가 들림 → 소리가 커지는 방향으로 이동
4. 화면 색감이 따뜻해짐 → 가까워지고 있음을 직감
5. 여동생 발견 → 안도감과 함께 클리어
6. 결과 화면에서 SNS 공유
```

---

## 4. 목표 및 성공 지표

### 4.1 제품 목표
- 5분 이내 플레이로 완결된 감정 경험 제공
- 바이럴 공유를 통한 자연스러운 확산

### 4.2 성공 지표 (MVP)

| 지표 | 목표값 |
|------|--------|
| 평균 클리어 시간 | 1~5분 |
| 클리어율 (중도 이탈 없이) | 70% 이상 |
| 재플레이율 | 1회 이상 재시도 |
| 공유율 | 클리어 후 1회 이상 공유 |

## 5. MVP 기능 명세

### 5.1 게임 흐름

```
[시작 화면] → [게임 플레이] → [클리어/종료 화면]
     ↓              ↓                  ↓
  설명 표시      탐색 진행         결과 + 공유
```

### 5.2 기능 상세

#### 5.2.1 시작 화면
- 게임 제목 표시
- 간단한 조작 설명 (WASD 또는 방향키)
- "시작" 버튼
- 배경: 가벼운 눈 파티클 효과

#### 5.2.2 게임 플레이

**이동 시스템**
- 1인칭 시점
- WASD / 방향키로 이동
- 마우스로 시점 회전 (또는 터치 스와이프)

**눈보라 파티클**
- 화면 전체를 덮는 눈 파티클
- 시야 거리 제한 (5~10m)
- 방향 감각 교란 효과

**여동생 스폰**
- 게임 시작 시 랜덤 위치에 스폰
- 플레이어로부터 일정 거리 이상 떨어진 곳

**거리 기반 오디오 (핵심 메카닉)**
- 여동생의 울음소리가 거리에 따라 변화
- 가까울수록: 볼륨 증가, 선명해짐
- 멀수록: 볼륨 감소, 흐릿해짐
- 3D 공간 오디오로 방향 힌트 제공

**화면 색감 힌트 (핵심 메카닉)**
- 멀 때: 차가운 푸른 톤
- 가까울 때: 따뜻한 톤으로 점진적 변화
- 색 변화는 미묘하게 (너무 명확하면 긴장감 감소)

**충돌 및 클리어**
- 여동생과 일정 거리 이내 접근 시 클리어
- 클리어 시 눈보라 멈춤 + 화면 밝아짐

**타이머**
- 플레이 시간 내부 기록 (화면에 표시 X)
- 클리어 시 결과 화면에 표시

#### 5.2.3 클리어/종료 화면
- "여동생을 찾았습니다" 메시지
- 클리어 시간 표시
- "다시 하기" 버튼
- "공유하기" 버튼 (Twitter/X, 링크 복사)

## 8. 리스크 및 대응

| 리스크 | 대응 방안 |
|--------|-----------|
| 너무 어려워서 이탈 | 색감 힌트 강도 조절, 거리에 따른 단계적 힌트 |
| 너무 쉬워서 재미 없음 | 눈보라 강도 조절, 스폰 거리 증가 |
| 모바일 조작 불편 | 가상 조이스틱 UI 추가 |
| 오디오 안 들림 (무음 모드) | 시작 전 "소리를 켜주세요" 안내 |

## 부록: 감정 곡선 설계

```
감정 강도
    │
 안도 │                              ╭───
    │                            ╱
 집중 │              ╭──────────╯
    │            ╱
 불안 │  ╭───────╯
    │╱
    └─────────────────────────────────── 시간
      시작    탐색 중    발견 직전   클리어
```

- **시작 (불안)**: 아무것도 안 보이는 눈보라, 방향 상실
- **탐색 (집중)**: 소리에 집중, 색감 변화 감지
- **발견 직전 (고조)**: 소리가 커짐, 색이 따뜻해짐
- **클리어 (안도)**: 눈보라 멈춤, 밝은 화면, 성취감
